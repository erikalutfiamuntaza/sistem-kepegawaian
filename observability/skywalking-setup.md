# Apache SkyWalking Setup

## Tujuan

Melakukan monitoring dan tracing terhadap aplikasi Sistem Informasi Kepegawaian.

## Langkah Implementasi

1. Menjalankan SkyWalking OAP Server menggunakan Docker.
2. Menjalankan SkyWalking UI menggunakan Docker.
3. Mengunduh dan memasang SkyWalking Java Agent.
4. Menjalankan backend dengan konfigurasi agent SkyWalking.
5. Memastikan service muncul pada dashboard SkyWalking.

## Hasil

SkyWalking berhasil menampilkan:

* Service topology
* Trace request
* Response time
* Monitoring performa aplikasi
